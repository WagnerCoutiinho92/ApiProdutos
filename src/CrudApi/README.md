# CRUD API .NET – SQL Server + Entity Framework Core

API RESTful completa utilizando **.NET 7**, **Entity Framework Core** e **SQL Server**.

## Tecnologias
- .NET 7
- ASP.NET Web API
- Entity Framework Core
- SQL Server
- AutoMapper
- Swagger

## Arquitetura
Controller → Service → Repository  
DTOs | Soft Delete | Paginação | Filtro Global

## Executar
dotnet restore
dotnet ef database update
dotnet run

Swagger:
https://localhost:5001/swagger
